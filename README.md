# 🎰 Slot Machine Game

ブラウザで動作するシンプルで楽しいスロットマシンゲーム

## 🚀 Quick Start

\`\`\`bash
npm install
npm run dev
\`\`\`

Open [http://localhost:3000](http://localhost:3000) to play!

## 📋 詳細仕様

完全な仕様書は [SPECIFICATION.md](./SPECIFICATION.md) をご覧ください。

## 🎮 Features

- 🎰 リアルなスロットマシン体験
- 💰 コイン管理システム  
- 🎯 多彩な勝利パターン
- 📱 レスポンシブデザイン
- 🔄 リセット機能

## 🛠️ Tech Stack

- React 19 / Next.js 15.3.4
- TypeScript
- Tailwind CSS
- Vercel (Deployment)
